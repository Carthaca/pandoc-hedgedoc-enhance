\subsection{Malformed Patterns}\label{malformed-patterns}

These should remain unchanged:

\begin{itemize}
\tightlist
\item
  Empty name: {[}name={]}
\item
  Unclosed bracket: {[}name=Someone
\item
  No opening bracket: name=Something{]}
\item
  Just brackets: {[}name{]}
\item
  Wrong format: {[}username=Bob{]}
\end{itemize}
